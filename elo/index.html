Some Commands

Use the Commands Link above for a full list of commands!

=register <name>

Register for the ELO Tournament in your server

This will allow your name to be updated with scores and so on

=SetAnnouncements

Make Game Announcements get sent to a specific channel

This will automatically @ all users in the game, show what team they are in, pick a random map and select a random host

=Game <lobby> <game> <team>

Automatically update team scores and wins/losses

Based on each users rank in game they will receive a certain amount of points for winning or lose a certain amount of points for losing a game

=addmap <maps>

Add Maps to a specific lobby

Want a lobby to only play certain maps? Just use =addmap to set these, you may also use =delmap <map> to remove a map and =maps to show all maps

=addrank <points> <@rank>

Add ranks to your server and give players goals to achieve

Users who reach the amount of points will receive this rank, if they drop below that amount of points they will lose it

=rankwinmodifier <@rank> <points>

Set the amount of points a user will receive if the have a certain rank and win

In this example ELO 50+ role will receive 100 points if they win whereas ELO Registered will only receive 25
You can also do this for the points per loss with
=ranklossmodifier <@rank> <points>

For the registered role, use the following:
=ModifyLoss - Sets the servers Loss amount
=ModifyWin - Sets the servers Win amount

Note

This is not a game, merely a game hosting/pairing bot, it makes teams for you to go play other games with, it's highly adaptable to cs-go, rainbow 6 siege, Warships & More
