Admin Commands

Game Commands

=Win - Add a win + win points for the specified users
=Lose - Add a loss to the specified users
=UndoGame - Undo the results of a previous game
=Game - Automatically update wins/losses for the selected team

Lobby Commands

=Createlobby - Initialise a lobby in the current channel
=RemoveLobby - Remove A Lobby
=ClearQueue - Clear All Players from The Queue
=AddMap - Add A Map
=DelMap - Delete A Map
=SetMaps - Set all maps for the current lobby
=NoPair - Disable the use of pairing features in the current queue
=ClearMaps - Clear all maps for the current lobby
=GameList - List previous games and their results

Management Commands

=DelUser <@user> - deletes a user from the server's registered list
=Registeruser <@user> - registers the user in the server
=Rename <@user> - Change a user's nickname
=ban <@user> - ban user from using Lobbies for specified anount of hours
=unban <@user> - Unban the specified user
=bans - List all bans in the server.
=addrank <@role> - set a role which people are given once reaching the specified points
=removerank <@role> - remove a role from the Ranks list
=removerank - remove a role from the Ranks list by ID
=RankWinModifier <@role> - allow a specific ranks points to be modified differently to server default
=RankLossModifier <@role> - allow a specific ranks points to be modified differently to server default
=ClearUsers - Deletes ALL user profiles in the server (SERVER OWNER ONLY)

Points Commands

=AddPoints - add points to the specified users
=DelPoints - remove points from the specified users
=SetPoints - set a user's exact points
=DelWin - remove one win from the specified users
=DelLose - remove a single loss from the specified users

 Setup Command

=Inisialise - Run this command to add your server to the serverlist
=SetAnnouncements - Set the current channel for game announcements
=SetNicknameFormat [ID] - Set how scores are displayed in player nicknames
=SetRegisterRole <@role> - Sets the role users will join when registering
=ToggleAutoRemove - Set if users are removed from the queue when going offline
=ToggleMultiQueue - Toggle wether users are able to join more than one queue at a time
=Premium - Upgrade the server to premium and increase the userlimit to unlimited
=SetAdmin <@role> - sets the configurable admin role
=SetMod <@role> - Sets the moderator role (point updating access)
=SetRegistermessage - sets the configurable registration message
=ModifyLoss - Sets the servers Loss amount
=ModifyWin - Sets the servers Win amount
=ScoreboardReset - Reset Points, Wins and Losses for all users in the server
=Blacklist - List blacklisted commands
=BlacklistAdd - Blacklist a command from all regular users.
=BlacklistDel - Remove a blacklisted command
=Server - Stats and info about the bot & current server
